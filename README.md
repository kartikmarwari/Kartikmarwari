### 🤖 Featured Project — IncidentMind
**AI-Powered Multi-Agent Root Cause Analysis System**
`Python` `FastAPI` `LangChain` `LangGraph` `FAISS` `Streamlit`

- Built an AI-driven incident analysis platform that automatically detects root causes from raw system logs using LLM-based causal reasoning and vector similarity search (FAISS).
- Engineered a multi-layer agent architecture (**Input → Preprocessing → Embedding → LLM Reasoning → Structured JSON Output**) with configurable LLM backends — OpenAI, Groq, Gemini, and Mistral.
- Implemented automated timeline generation and causal chain analysis (e.g., *Deploy → Latency Spike → DB Timeout → Crash*) with confidence scoring for each predicted root cause.
- Designed a Streamlit dashboard for real-time incident visualization with structured JSON output integration into external DevOps pipelines.
